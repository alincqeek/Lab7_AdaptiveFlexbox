# Лабораторная работа №7. Адаптивная вёрстка: Flexbox

В этой лабораторной работе вы познакомитесь с основами адаптивной вёрстки с использованием технологии **Flexbox**. Вы научитесь создавать гибкие, отзывчивые макеты страниц, которые корректно отображаются на различных устройствах.

## Основные этапы
1. Подготовка проекта
    * Создание рабочей директории
    * Инициализация Git + подключение к GitHub
    * Создание базовых файлов: index.html, style.css
2. Теория Flexbox
    * Контейнер:
    `display: flex;
flex-direction: row | column;
justify-content; /* главная ось */
align-items;     /* поперечная ось */
flex-wrap;
gap;`

    * Элементы:
    `flex-grow / flex-shrink;
flex-basis;
flex: grow shrink basis;
align-self;`
3. Практические примеры
* Навигационное меню — горизонтальное выравнивание
 * Карточки товаров — адаптивная сетка
* Центрирование — точное позиционирование по центру
4. Доработка стилей:
* Оформление меню, кнопок, карточек
* Эффекты наведения (:hover)
* Адаптивность через flex-wrap
